## 📖 Instalação e Uso

1. Clone o repositório:
    ```bash
    git clone https://github.com/HarisonRios/LabTricolor
    ```

2. Entre no diretorio Site
   ```bash
    cd Site
    ```
3. Instale as Dependências
   ```bash
    npm i ou npm install
    ```
4. inicialize o Serivodor Local - app.js
   ```bash
    npm start
    ```
